# 🚀 立即部署到 Vercel

## 最簡單的方式：網頁操作（推薦！）

### ⚡ 5 分鐘完成部署

1. **開啟瀏覽器，前往：** https://vercel.com/new

2. **登入 Vercel：**
   - 點擊「Continue with GitHub」
   - 使用您的 GitHub 帳號（`RheC1e`）登入

3. **匯入專案：**
   - 搜尋或選擇：`RheC1e/rhema-pwa-demo`
   - 點擊「Import」

4. **部署設定（自動偵測，無需修改）：**
   - Framework: **Vite** ✅
   - Build Command: `npm run build` ✅
   - Output Directory: `dist` ✅
   - 直接點擊「Deploy」！

5. **等待 2-3 分鐘，完成！** 🎉
